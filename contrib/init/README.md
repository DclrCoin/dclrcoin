Sample configuration files for:
```
SystemD: dclrcoind.service
Upstart: dclrcoind.conf
OpenRC:  dclrcoind.openrc
         dclrcoind.openrcconf
CentOS:  dclrcoind.init
macOS:   org.dclrcoin.dclrcoind.plist
```
